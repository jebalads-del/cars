# سويفت موتورز - منصة بيع السيارات الفاخرة

موقع احترافي لبيع السيارات الفاخرة مع لوحة تحكم متكاملة لإدارة المخزون.

## المميزات الرئيسية

### الواجهة الأمامية (Frontend)
- **الصفحة الرئيسية**: عرض احترافي مع قسم Hero جذاب وعرض السيارات المميزة
- **معرض السيارات**: صفحة تفاعلية بقائمة كاملة للسيارات مع خيارات البحث والتصفية
- **تفاصيل السيارة**: صفحة ديناميكية لكل سيارة مع جميع المواصفات والميزات
- **تصميم متجاوب**: يعمل بشكل مثالي على جميع الأجهزة (موبايل، تابلت، ديسكتوب)

### لوحة التحكم (Admin Dashboard)
- **إدارة المخزون**: إضافة وتعديل وحذف السيارات
- **إدارة العملات**: اختيار العملة الخليجية لكل سيارة
- **ادارة الحالة**: تحديث حالة السيارة (متاحة، مباعة، قيد الانتظار)
- **نظام المصادقة**: كلمات مرور آمنة وجلسات المستخدم

### اللغة والعملات
- **اللغة العربية**: واجهة كاملة باللغة العربية مع اتجاه RTL
- **العملات الخليجية**: 
  - الدينار الكويتي (KWD) - العملة الافتراضية
  - الدرهم الإماراتي (AED)
  - الريال السعودي (SAR)
  - الدينار البحريني (BHD)
  - الريال العماني (OMR)
  - الريال القطري (QAR)

## الهيكل التقني

### التكنولوجيا المستخدمة
- **Next.js 16**: إطار عمل React متقدم
- **TypeScript**: لكتابة كود آمن وموثوق
- **Tailwind CSS**: للتصميم الحديث والمتجاوب
- **Lucide Icons**: أيقونات جميلة وسهلة الاستخدام
- **localStorage**: لتخزين بيانات السيارات محلياً

### البنية الملفية

```
/app
  /admin        - لوحة التحكم
  /cars
    /[id]       - صفحة تفاصيل السيارة
    /page.tsx   - صفحة معرض السيارات
  /layout.tsx   - التخطيط الرئيسي
  /page.tsx     - الصفحة الرئيسية

/components
  /header.tsx     - شريط التنقل
  /car-card.tsx   - بطاقة السيارة

/lib
  /types.ts       - الأنواع والترجمة
  /car-data.ts    - بيانات السيارات
  /currency.ts    - دوال تحويل العملات
  /utils.ts       - دوال مساعدة
```

## كيفية الاستخدام

### تشغيل الموقع محلياً

```bash
# تثبيت المتطلبات
pnpm install

# تشغيل خادم التطوير
pnpm dev

# الموقع سيكون متاحاً على http://localhost:3000
```

### إضافة سيارة جديدة

1. اذهب إلى `/admin`
2. سجل الدخول بـ:
   - Username: `admin`
   - Password: `swift2024`
3. اضغط على "إضافة سيارة جديدة"
4. ملأ البيانات واختر العملة المناسبة
5. اضغط "حفظ"
